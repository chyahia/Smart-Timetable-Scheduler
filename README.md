# 📅 موزع الجداول الجامعية الذكي (Smart Timetable Scheduler)

نظام متكامل ومتقدم لإنشاء وتوزيع الجداول الزمنية الجامعية آلياً باستخدام خوارزميات البحث المتقدمة. تم تصميم هذا النظام لإنهاء معاناة الجدولة اليدوية، مع مراعاة أدق التفاصيل والقيود الخاصة بالأساتذة، القاعات، والمستويات الدراسية.

## ✨ المميزات الرئيسية والابتكارات

* **محرك خوارزمي هجين (Hybrid Engine):**
  * **البناء السريع:** تأسيس الجدول المبدئي في ثوانٍ باستخدام الخوارزمية الطماعة (Greedy Algorithm).
  * **التحسين المتقدم:** تلميع الجدول وحل التعارضات باستخدام مزيج من خوارزميات (Tabu Search)، (LNS)، و (VNS).

* **أداة ضغط الجداول وسد الفجوات (Refinement):**
  * نظام ذكي يتدخل بعد التوزيع بـ 3 مستويات (بسيط، متوازن، عميق).
  * وظيفته تفريغ الحصص المتأخرة، إزالة الفراغات بين الحصص، وهدم وبناء جداول الأساتذة المعقدة لضمان أقصى درجات الراحة.

* **إدارة قيود مرنة وصارمة (Constraint Management):**
  * تحكم كامل بأوزان القيود (إلزامية، مهمة، تفضيلية).
  * إدارة أيام عمل الأساتذة، منع توالي القاعات الكبيرة، تقييد العمل يوم السبت، وإدارة القيود المكانية والزمنية المعقدة.

* **واجهة مستخدم تفاعلية (UI/UX):**
  * **بث حي للعمليات:** شاشة تتبع (Terminal) داخل المتصفح تظهر تفاصيل حركة الخوارزميات وصناعة القرار خطوة بخطوة.
  * **آلة الزمن (ذاكرة حية):** ميزة حفظ واستعادة النتائج (1 و 2) في المتصفح للمقارنة بين التوزيعات دون فقدانها.
  * **تصدير احترافي:** استخراج الجداول النهائية (مستويات، أساتذة، قاعات فارغة) بصيغ Excel و PDF.

## 🛠️ التقنيات المستخدمة
* **الخلفية (Backend):** Python, Flask
* **قاعدة البيانات (Database):** SQLite
* **الواجهة (Frontend):** HTML5, CSS3, JavaScript (Vanilla)
